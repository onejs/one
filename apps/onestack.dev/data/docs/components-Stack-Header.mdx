title: Stack.Header
---

A declarative JSX API for configuring native stack headers. This is an alternative to the traditional `options` prop approach.

## Basic Usage

```tsx
import { Stack } from 'one'

export default function Layout() {
  return (
    <Stack>
      <Stack.Screen name="index">
        <Stack.Header>
          <Stack.Header.Title>Home</Stack.Header.Title>
        </Stack.Header>
      </Stack.Screen>
    </Stack>
  )
}
```

## Large Title with Blur (iOS)

The classic iOS large title pattern:

```tsx
<Stack.Screen name="feed">
  <Stack.Header blurEffect="regular">
    <Stack.Header.Title large>Feed</Stack.Header.Title>
  </Stack.Header>
</Stack.Screen>
```

For iOS, pair this with `contentInsetAdjustmentBehavior="automatic"` on your ScrollView.

## Custom Header Buttons

```tsx
<Stack.Screen name="profile">
  <Stack.Header>
    <Stack.Header.Title>Profile</Stack.Header.Title>
    <Stack.Header.Right asChild>
      <Pressable onPress={openSettings}>
        <SettingsIcon />
      </Pressable>
    </Stack.Header.Right>
  </Stack.Header>
</Stack.Screen>
```

## Search Bar

```tsx
<Stack.Screen name="search">
  <Stack.Header>
    <Stack.Header.Title>Search</Stack.Header.Title>
    <Stack.Header.SearchBar
      placeholder="Search..."
      onChangeText={handleSearch}
    />
  </Stack.Header>
</Stack.Screen>
```

## Components

### Stack.Header

Main container for header configuration.

| Prop | Type | Description |
|------|------|-------------|
| `hidden` | `boolean` | Hide the header |
| `asChild` | `boolean` | Render children as custom header |
| `blurEffect` | `BlurEffect` | iOS blur effect type |
| `style` | `StyleProp` | Background and shadow style |
| `largeStyle` | `StyleProp` | Large title mode style |

### Stack.Header.Title

```tsx
<Stack.Header.Title large>My Title</Stack.Header.Title>
```

| Prop | Type | Description |
|------|------|-------------|
| `children` | `string` | Title text |
| `large` | `boolean` | Enable iOS large title |
| `style` | `StyleProp` | Title text style |
| `largeStyle` | `StyleProp` | Large title text style |

### Stack.Header.Left / Stack.Header.Right

Custom content for header sides. Requires `asChild` to render.

```tsx
<Stack.Header.Right asChild>
  <Button>Action</Button>
</Stack.Header.Right>
```

### Stack.Header.BackButton

Configure back button appearance.

| Prop | Type | Description |
|------|------|-------------|
| `children` | `string` | Back button title |
| `hidden` | `boolean` | Hide back button |
| `displayMode` | `string` | Display mode |

### Stack.Header.SearchBar

Integrated search bar. Accepts all `SearchBarProps` from react-native-screens.

## Mixing with Options

Composition API and options work together:

```tsx
<Stack.Screen
  name="detail"
  options={{ animation: 'slide_from_right' }}
>
  <Stack.Header>
    <Stack.Header.Title large>Detail</Stack.Header.Title>
  </Stack.Header>
</Stack.Screen>
```

See the [React Navigation Native Stack docs](https://reactnavigation.org/docs/native-stack-navigator#options) for all available options.
